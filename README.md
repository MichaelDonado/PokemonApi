### Pokedex Flutter/MobX

### Features

- List of Pokemons
- Pantalla de detalles de Pokémon (detalles, estado, evoluciones)

### Dev Tools
- Aplicación realizada con gestión de estado mediante MobX y gestión de instancias mediante la biblioteca GetIt.
